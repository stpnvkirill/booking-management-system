## Что можно использовать для замены/дополнения Tailwind:

 - DaisyUI - Предоставляет готовые компоненты с возможностью легкого тематического оформления (темная тема, контрастность), которые работают поверх Tailwind
 - FlowBite - Большая библиотека с компонентами, секциями и даже готовыми шаблонами
 - HyperUI - библиотека бесплатных компонентов  
 - PrelineUI - готовые шаблоны и функциональные плагины без необходимости в JS, как и FlyonUI
 - FlyonUI - Очень схож с предыдущим
 - PandaCSS - Инструментарий для генерации CSS в стиле Tailwind
 - Headless UI - набор не стилизованных компонентов tailwind
